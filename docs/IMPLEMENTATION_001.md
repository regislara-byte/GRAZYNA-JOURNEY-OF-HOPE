## Project Vision

Create a respectful, family-controlled digital documentary website that tells Grażyna Nowak's story, documents recovery milestones, hosts future vlog updates, and directs visitors to the official Siepomaga fundraiser.

This project is not a fundraising platform.

It is a digital story archive and support journal.

Provide a central location where family, friends, supporters, and visitors can:

* Learn about Grażyna
* Follow recovery updates
* Watch future vlog content
* Share messages of hope
* Access the official Siepomaga fundraiser
